Jam tools

These tools works great for game jams and are mostly free.

Tools for making computer games

  • 3D (or 2D)

    • Unity (C#)
    • Godot (GDScript: Python-like syntax)
    • Unreal Engine (C++/Visual Scripting)

  • 2D

    • Construct (Visual Scripting)
    • Pygame (Python)
    • Löve (Lua)
    • Haxeflixel (Haxe)
    • Scratch (Visual Scripting)

Tools for making graphics

  • Blender 3D (3D modelling and more)
  • MagicaVoxel (Voxel editor)
  • Krita (2D drawing and animation)
  • MakeHuman (3D-models of humans)
  • Simplygon (3D-model optimization)
  • Mixamo (Autorig and animate humanoid 3D characters)

Tools for making sound effects

  • bfxr (Online old school sound effect generator)
  • Sunvox (Modular synthesizer / tracker for music)
  • BeepBox (Online chiptune music tracker)
  • ChipTones (Old school sound effect generator)

Free game resources

  • OpenGameArt.org (Public domain and creative commons game assets)
  • Itch.io Game Assets
  • HDRI Haven (HDR / Environment Maps)
  • cgbookcase.com (Textures)
  • Texture Haven (Textures)
  • CC0 Textures (Textures)
  • textures.com (Textures)
  • Dev Assets
  • game-icons.net (Icons)
  • Gameplay Design Patterns

Tools for keeping track of work (version control)

  • Hosts

    • GitLab
    • GitHub
    • Linköping University's GitLab

  • Clients

    • Git
    • SourceTree (frontend for Git)
